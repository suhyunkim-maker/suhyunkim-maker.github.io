## 1. 분석 개요 및 라이브러리 로드

부동산 가격은 면적, 입지, 브랜드, 층수, 건축년도 등 다양한 요인에 의해 복합적으로 결정됩니다. 단순하게 전체 데이터를 비교하면 면적이 넓어서 비싼 것인지, 입지가 좋아서 비싼 것인지 구분하기가 모호합니다. 따라서 이번 분석에서는 **최대한 다른 변수를 통제(Control)**하고 개별 요인이 단위면적당 가격에 미치는 영향을 검증하고자 합니다.

먼저 데이터 분석에 필요한 기본적인 라이브러리를 로드하고 한글 폰트를 설정합니다.


## 2. 엄격한 변수 통제 (Variable Control)

노이즈를 줄이고 동일 조건에서의 비교를 위해 다음과 같이 필터링을 진행하여 분석 대상을 통제했습니다.

1. **정상 거래만 포함**: 취소일(`취소일`)이 존재하지 않는 거래만 선별
2. **거래 형태 제한**: 분양권 거래(`권리구분 != '분양권'`) 제외
3. **중개 형태 제한**: 직거래를 제외한 중개거래(`신고구분 == '중개거래'`)만 포함
4. **시기 통제**: 2025년도 한 해 동안 발생한 계약 자료
5. **면적 통제**: 전용면적 **84㎡ ~ 89㎡** 사이 (소위 국민평형 84㎡ 전후 기준)로 제한
6. **지역 통제**: 서울시 내에서 비교적 아파트 단지가 밀집되어 있고 거래량이 풍부한 **'노원구'**로 한정

```python
# 필터링 적용
df = df[df['취소일'].isna()]
df = df[df['권리구분'] != '분양권']
df = df[df['신고구분'] == '중개거래']
df = df[df['계약일'].between(20250101, 20260101)]  # 2025년 거래 자료
df = df[df['건물면적(㎡)'].between(84, 89)]

print("필터링 후 분석 대상 거래 건수:", len(df1))
```

> ```text

## 4. 아파트 단지(건물명)별 단위가격 분석 및 이상치 제거

거래량이 너무 적은 단지들은 통계적 의미가 떨어지므로, 노원구 내에서 2025년에 가장 거래가 활발했던 **상위 20개 아파트 단지**를 선별했습니다.

```python
# 거래량 상위 20개 단지명 추출
N = 20
top20_buildings = df1['건물명'].value_counts().head(N).index
df_topN = df1[df1['건물명'].isin(top20_buildings)]
```

### 4.1 단지별 단위가격 이상치 제거 (IQR 기준)
특정 단지 내에서 특수 거래나 직거래 착오 등으로 인해 발생한 극단적인 가격 데이터를 정제하기 위해, 각 아파트 단지별로 사분위수 범위(IQR) 방식을 사용해 **이상치를 정제**했습니다.

```python
# 1. 이상치 판단 함수 정의 (Q1 - 1.5*IQR ~ Q3 + 1.5*IQR)
def remove_series_outliers(series):
    q1 = series.quantile(0.25)
    q3 = series.quantile(0.75)
    iqr = q3 - q1
    lower_bound = q1 - 1.5 * iqr
    upper_bound = q3 + 1.5 * iqr
    return (series >= lower_bound) & (series <= upper_bound)

# 2. 그룹별(단지별)로 이상치 마스크 생성 및 정렬
valid_mask = df_topN.groupby('건물명')['단위가격'].apply(remove_series_outliers, include_groups=False)
valid_mask = valid_mask.reset_index(level=0, drop=True).sort_index()

# 3. 데이터 필터링
df_filtered_outliers = df_topN[valid_mask]

print("이상치 제거 전 거래 건수:", len(df_topN))
print("이상치 제거 후 최종 거래 건수:", len(df_filtered_outliers))
```
> **출력 결과:**
> ```text
> 이상치 제거 전 거래 건수: 477건
> 이상치 제거 후 최종 거래 건수: 461건
> ```

### 4.2 이상치 정제 후 단지별 단위가격 분포 시각화 (Violinplot)
이상치가 성공적으로 제거된 상위 20개 단지의 단위가격 분포를 **바이올린 플롯**으로 시각화했습니다.

```python
topN_counts = df_filtered_outliers['건물명'].value_counts().head(N)
```

## 5. 가설 검정 및 통계 분석 (Hypothesis Testing)

단지별로 평균 단위가격에 눈에 띄는 차이가 존재하는 것처럼 보이지만, 이것이 통계적으로 유의미한지 검증하기 위해 **일원배치 분산분석(One-way ANOVA)**과 **피어슨 상관분석**을 실시합니다.

### 5.1 아파트 단지명(건물명)별 단위가격 차이 검정
* **귀무가설 ($H_0$)**: "상위 20개 아파트 단지의 평균 단위가격은 모두 같다."
* **대립가설 ($H_1$)**: "상위 20개 아파트 단지 중 최소한 한 단지의 평균 단위가격은 다른 단지들과 다르다."

```python
import scipy.stats as stats

# 그룹별 데이터 추출
target_buildings = list(summary_agg_sorted.index)
group_data = [
    df_filtered_outliers[df_topN['건물명'] == building]['단위가격'] 
    for building in target_buildings
]

# One-way ANOVA 수행
f_statistic, p_value = stats.f_oneway(*group_data)

print(f"F-통계량 (F-statistic) : {f_statistic:.4f}")
print(f"유의확률 (p-value)      : {p_value:.4e}")
```
> **검정 결과:**
> * **F-통계량**: 311.2309
> * **p-value**: 1.5049e-241

* **해석**: 유의확률(p-value)이 유의수준 0.05보다 훨씬 작은 거의 0에 가깝기 때문에 귀무가설을 기각합니다. 즉, **아파트 단지명(브랜드, 구체적인 입지 등)에 따른 가격 차이는 통계적으로 극히 유의미하게 존재함**을 알 수 있습니다.

---

### 5.2 건축년도(연식)와 평균 단위가격의 상관분석
신축일수록 가격이 비쌀 것이라는 통념이 있습니다. 단지별 평균 가격과 해당 단지의 건축년도 간의 선형 상관관계가 존재하는지 파악하기 위해 피어슨 상관분석을 시행합니다.

```python
correlation, p_value = stats.pearsonr(building_summary['건축년도'], building_summary['금액_평균'])

print(f"피어슨 상관계수 (r): {correlation:.4f}")
print(f"유의확률 (p-value) : {p_value:.4e}")
```
> **검정 결과:**
> * **피어슨 상관계수 (r)**: 0.3448
> * **p-value**: 0.1365 (1.3655e-01)

* **해석**: p-value가 0.05보다 크므로 상관관계가 통계적으로 유의미하지 않습니다. 즉, **노원구 내 동일 평형 아파트들에 있어서 단순히 건축년도만으로는 단지의 평균 가격을 설명하기 어렵다**는 결론을 내릴 수 있습니다. 이는 건축연식 외에 학군이나 역세권과 같은 입지 프리미엄이 가격에 지대한 영향을 주기 때문으로 해석됩니다 (예: 1990년대 중반 연식임에도 대장 역할을 하는 중계동 학군지 단지들).

### 5.3 법정동명에 따른 평균 단위가격 차이 검정 (ANOVA)
노원구 내에서 법정동(예: 중계동, 상계동, 월계동, 하계동, 공릉동)에 따라 평균적인 가격 격차가 유의미한지 알아보기 위해 ANOVA 검정을 실시합니다.

```python
# 법정동별 그룹 데이터 수집
groups = [group['금액_평균'].values for name, group in building_summary.groupby('법정동명')]

# ANOVA 검정
f_stat, p_value = stats.f_oneway(*groups)

print(f"F-통계량 (F-statistic): {f_stat:.4f}")
print(f"유의확률 (p-value)    : {p_value:.4f}")
```
> **검정 결과:**
> * **F-통계량**: 1.4285
> * **p-value**: 0.2727

* **해석**: p-value가 0.05보다 크므로 귀무가설을 기각할 수 없습니다. 즉, **노원구 내에서 법정동명에 따른 아파트들의 평균 가격 차이는 통계적으로 유의미하지 않습니다.** 동별 차이보다는, **개별 아파트 단지가 갖는 구체적인 브랜드 및 초입지 특성(지하철 접근성, 단지 규모, 초등학교 품은 아파트 여부 등)**이 가격 결정에 훨씬 크게 작용함을 시사합니다.

## 6. 결론 요약 및 시사점

1. **단지명(개별 브랜드와 입지)이 가장 결정적인 요인**: 동일 자치구(노원구) 내에서 면적(84~89㎡)과 거래 형태(중개 거래) 등을 모두 통제한 결과, 아파트 가격 차이를 결정하는 가장 뚜렷하고 유의미한 요인은 **'아파트 단지명'**이었습니다 (ANOVA 유의확률 ~ 0).
2. **연식의 한계**: 단순히 최근에 지어진 신축 아파트라고 해서 무조건 가격이 높게 결정되는 것은 아닙니다. 구축 아파트라도 우수한 인프라와 강력한 학군(예: 중계동 학군지 아파트)을 갖췄다면 신축 프리미엄을 상회하는 높은 가치를 가집니다.
3. **법정동명보다 마이크로 입지**: 행정 구역(동) 단위의 구분은 단지 간의 평균 가격 차이를 통계적으로 잘 구별하지 못했습니다. 결국 같은 동 내에서도 어떤 단지인가에 따라 가치 편차가 매우 큽니다.

데이터 분석과 통계 검정을 통해, 부동산을 구매하거나 투자할 때는 행정동이나 신축 여부라는 거시적 요소보다 **"개별 단지가 가진 구체적이고 마이크로한 입지 가치(학군, 역세권, 대단지)"**를 꼼꼼히 평가해야 한다는 점을 데이터로 확인할 수 있었습니다.
